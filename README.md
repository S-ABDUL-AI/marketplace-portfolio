## Data Notice

All tools use fully synthetic data generated from calibrated economic
models and published marketplace research. No real user data,
proprietary business data, or scraped data is used.

---

## Run Locally

```bash
git clone https://github.com/S-ABDUL-AI/marketplace-portfolio.git
cd marketplace-portfolio
pip install -r requirements.txt
streamlit run app.py
```
